## Getting Started

### Prerequisites

- Python 3.8+  
- OpenAI API key ([Get one here](https://platform.openai.com/signup))  
- OpenWeatherMap API key ([Get one here](https://openweathermap.org/api))  

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/hamburgem/Weatherly-ai-chatbot.git
   cd Weatherly-ai-chatbot

2. Install dependencies:
    pip install -r requirements.txt

3. Set environment variables for your API keys:
    export OPENAI_API_KEY="your_openai_api_key_here"
    export OPENWEATHER_API_KEY="your_openweathermap_api_key_here"

4.Run the chatbot:
    python main.py
